ขนาดอัตราส่วนของกระดาษ A B C ANSI

📅 6 ก.พ. 2566⏱ อ่าน 5 นาที🏷 กระดาษ

ขนาดกระดาษตามมาตรฐาน ISO 216 แบ่งออกเป็น 4 ชุด ดังนี้

ชุด A : มีขนาดเริ่มต้นที่ A0 มีพื้นที่ 1 ตารางเมตร ซึ่งเมื่อตัดหรือพับครึ่งด้านที่ยาวที่สุด
ก็จะได้กระดาษขนาด A1 จำนวน 2 แผ่น และสามารถแบ่งเช่นนี้ไปเรื่อยๆ
จนได้กระดาษขนาด A10 ที่มีขนาดเล็กที่สุด

โดยกระดาษ A0 จำนวน 1 แผ่นใหญ่
จะสามารถตัดแบ่งเป็นกระดาษ A4 ได้มากถึง 16 แผ่น

โดยขนาดของกระดาษ 2A-A10 เท่ากับ
2A = 1189 x 1682 mm.
A0 = 841 x 1189 mm.
A1 = 594 x 841 mm.
A2 = 420 x 594 mm.
A3 = 297 x 420 mm.
A4 = 210 x 297 mm.
A5 = 148 x 210 mm.
A6 = 105 x 148 mm.
A7 = 74 x 105 mm.
A8 = 52 x 74 mm.
A9 = 37 x 52 mm.
A10 = 26 x 37 mm.

ชุด B : จะใช้หลักการแบ่งกระดาษและใช้อัตราส่วนสัมพันธ์เหมือนกับซีรีส์ A (1:√2)
ต่างกันที่กระดาษ B0 มีพื้นที่เพิ่มขึ้นมาอีกเป็น 1.414 ตารางเมตร

B0 = 1000 x 1414 mm.
B1 = 707 x 1000 mm.
B2 = 500 x 707 mm.
B3 = 353 x 500 mm.
B4 = 250 x 353 mm.
B5 = 176 x 250 mm.
B6 = 125 x 176 mm.
B7 = 88 x 125 mm.
B8 = 62 x 88 mm.
B9 = 44 x 62 mm.
B10 = 31 x 44 mm.

ชุด C : รหัสชุดนี้มีไว้ใช้กำหนดขนาดของซองใส่เอกสาร ที่เลขเดียวกัน
โดยรหัสของ C จะใหญ่กว่าของ A แต่เล็กกว่าของ B
ดังนั้น กระดาษจดหมาย A4 จะสามารถใส่ลงในซองขนาด C4
และซองขนาด C4 ก็จะใส่ลงในซอง B4 ได้พอเหมาะ

C0 = 917 x 1297 mm.
C1 = 648 x 917 mm.
C2 = 458 x 648 mm.
C3 = 324 x 458 mm.
C4 = 229 x 324 mm.
C5 = 162 x 229 mm.
C6 = 114 x 162 mm.
C7 = 81 x 114 mm.
C8 = 57 x 81 mm.
C9 = 40 x 57 mm.
C10 = 28 x 40 mm.

ชุด ANSI : ขนาดกระดาษมาตรฐานอเมริกา นิยมใช้กันในประเทศอเมริกาและใช้เป็นขนาดกระดาษเขียนแบบสำหรับสถาปนิกและวิศวกร

ANSI E = 864 x 1118 mm.
ANSI D = 559 x 864 mm.
ANSI C = 432 x 559 mm.
ANSI B = 279 x 432 mm.
ANSI A = 216 x 279 mm.

นอกจากนี้ ยังมีขนาดกระดาษในมาตรฐานอื่นๆ อีก เช่น ขนาดกระดาษมาตรฐาน JIS ของประเทศญี่ปุ่น และ SIS ของประเทศสวีเดน เป็นต้น

บทความเกี่ยวกับสิ่งพิมพ์อื่นๆ

  • ขนาดของกระดาษตามระบบ ISO
  • ความแตกต่างระหว่างโรงพิมพ์ OFFSET และ โรงพิมพ์ DIGITAL
  • ระบบสี CMYK กับ RGB ต่างกันอย่างไร
  • ส่ง FILES ARTWORK ให้โรงพิมพ์อย่างไร ให้ไม่มีปัญหา
  • ชนิดของกระดาษ ที่ใช้ในงานพิมพ์
  • ใช้โปรแกรมไหน ทำ Artwork ส่งโรงพิมพ์ดี
  • การเคลือบสิ่งพิมพ์ (Coating Method) แต่ละชนิดมีอะไรบ้าง
  • นามสกุลไฟล์ ที่ใช้ในงานพิมพ์แต่ละชนิดแตกต่างกันยังไง
  • กระดาษที่นิยมนำมาทำถุงมีกี่ชนิด
  • กระดาษที่นิยมนำมาผลิตกล่องมีกี่ชนิด
  • สื่อสิ่งพิมพ์โฆษณากระดาษ โบร์ชัวร์ ใบปลิว มีกี่ชนิด อะไรบ้าง
  • การเข้าเล่มหนังสือมีทั้งหมดกี่ชนิด

  • ท่ามกลางเทรนด์ของสื่อดิจิตอล ทำไมสื่อสิ่งพิมพ์ถึงยังมีความจำเป็นอยู่
  • วิธีการออกแบบสื่อสิ่งพิมพ์ให้มีประสิทธิภาพ
  • วิธีการสร้างแบรนด์ให้แข็งแกร่งโดยใช้สื่อสิ่งพิมพ์
  • ปัญหาที่พบบ่อยในงานสิ่งพิมพ์
  • ขั้นตอนหลักๆ ในการทำสิ่งพิมพ์ประเภทต่างๆ
  • เทคนิคการพิมพ์แบบพิเศษ
  • ความรู้เกี่ยวกับทฤษฎีการผสมสี CMYK
  • ชนิดของกระดาษใช้ทําบิลมีอะไรบ้าง
  • สาเหตุของปัญหา รอยขีดข่วน บนงานพิมพ์เกิดจากอะไร
  • คำแนะนำในการออกแบบ ฟอนต์ สี และการจัดรูปแบบของสื่อสิ่งพิมพ์

  • การใช้ AR ในอุตสาหกรรมการพิมพ์
  • พลังของบรรจุภัณฑ์ทางการตลาด
