ส่งไฟล์ยังไงไม่ให้เพี้ยน — 5 ความผิดพลาดที่ทำให้งานพิมพ์ออกมาไม่ตรงปก (และวิธีป้องกัน)

📅 26 มิ.ย. 2569⏱ อ่าน 8 นาที🏷 ไฟล์ & Artwork

1. ส่งไฟล์ RGB แทน CMYK — สีเพี้ยนทันที

จอคอมแสดงสีในระบบ RGB แต่เครื่องพิมพ์ทำงานด้วยหมึก CMYK ทั้งสองระบบมีสีที่ต่างกัน สีสดบนจออย่างฟ้าจัด เขียวนีออน หรือม่วง พอพิมพ์ออกมามักหม่นลงหรือเปลี่ยนโทนไปเลย

วิธีป้องกัน: ตั้งค่า Color Mode เป็น CMYK ตั้งแต่เปิดไฟล์ใหม่ ไม่ใช่แปลงทีหลัง
ใน Illustrator หรือ InDesign เช็กที่ Document Setup
ใน Photoshop เช็กที่ Image > Mode

2. ไม่ทำ Bleed (เผื่อตัดตก) — ขอบขาวโผล่หลังตัด

เครื่องตัดกระดาษไม่ได้ตัดตรงร้อยเปอร์เซ็นต์ มีการเหลื่อมเล็กน้อยเสมอ ถ้าสีหรือภาพในไฟล์ชิดขอบพอดี พอตัดออกมาจะเห็นขอบกระดาษขาวทันที Bleed คือพื้นที่กันชนที่ต้องยืดสีหรือพื้นหลังออกจากขอบจริงอีก 3 มิลลิเมตร

วิธีป้องกัน: ทุกงานที่มีสีหรือภาพพาดถึงขอบ ให้ตั้ง Bleed และยืดพื้นหลังให้เต็ม Bleed Area เสมอ เพื่อเป็นการเผื่อตัดตก

3. ความละเอียดต่ำเกินไป (72 dpi แทน 300 dpi) — ภาพเบลอ

ภาพที่คมชัดบนจอไม่ได้แปลว่าจะคมชัดเมื่อพิมพ์ จอแสดงผลใช้ 72-96 dpi แต่งานพิมพ์ต้องการอย่างน้อย 300 dpi ที่ขนาดจริงของงาน ภาพจากเว็บหรือ Facebook มักละเอียดไม่พอ พิมพ์ออกมาจะเบลอและเห็นพิกเซล

วิธีป้องกัน: เช็ก Resolution ก่อนวางภาพในไฟล์งานทุกครั้ง ใน Photoshop ดูได้ที่ Image > Image Size ถ้าต่ำกว่า 300 dpi ให้หาไฟล์ต้นฉบับที่ใหญ่กว่า หรือใช้ภาพ Vector แทน

4. Font เปลี่ยนเองเมื่อเปิดบนเครื่องอื่น — Layout พังทั้งหน้า

ไฟล์ที่ส่งไปไม่ได้พาฟอนต์ไปด้วยโดยอัตโนมัติ เมื่อโรงพิมพ์เปิดบนเครื่องที่ไม่มีฟอนต์ตัวเดียวกัน ระบบจะเปลี่ยนฟอนต์เป็นตัวอื่น หรือเด้งเป็นภาษาต่างดาว และ Layout จะพังทันที

วิธีป้องกัน มีสามทาง:
1) Outline Font — แปลงตัวอักษรให้กลายเป็น Vector Path ใน Illustrator ใช้ Type > Create Outlines ปลอดภัยที่สุดแต่แก้ข้อความไม่ได้อีก ควรเก็บไฟล์ต้นฉบับไว้ก่อน
2) ถ้าไม่ Create Outlines ให้ส่งฟอนต์ทั้งหมดที่ใช้มาด้วย เพื่อให้โรงพิมพ์มีฟอนต์ครบ Layout ไม่เสียหาย
3) Embed Font ตอน Export เป็น PDF เลือก Standard PDF/X-1a หรือ PDF/X-4 และเปิด Embed All Fonts — สำหรับฟอนต์ภาษาไทยเรื่องนี้สำคัญมาก

5. ไม่ตรวจ Proof ก่อนพิมพ์จริง — พลาดแล้วพลาดเลย

งานพิมพ์ไม่ใช่ไฟล์ดิจิทัลที่กด Undo ได้ พิมพ์ออกมาแล้วจบ เราเจอกรณีเบอร์โทรผิด โลโก้ขาดหาย ที่อยู่ไม่ถูก หรือสีโลโก้เพี้ยน ทั้งหมดนี้รู้เมื่อรับงานไปแล้วและต้องพิมพ์ใหม่ทั้งหมด

วิธีป้องกัน: ก่อนอนุมัติพิมพ์ทุกครั้ง เช็กว่าข้อความถูกต้องครบถ้วน เบอร์โทร อีเมล เว็บไซต์ถูกต้อง สีโลโก้ตรงกับ Brand Guide ขนาดของงานถูกต้อง อ่านทวนอีกรอบและให้คนที่ไม่ได้ทำงานนี้ช่วยอ่านทวนด้วย เพราะตาเราเองมักมองข้ามสิ่งที่คุ้นเคย

ความผิดพลาดเหล่านี้ป้องกันได้ทั้งหมดก่อนส่งไฟล์ เพื่อความมั่นใจ ตรวจเช็คความถูกต้องทั้งหมดก่อนส่งไฟล์ หรือให้ทางโรงพิมพ์ช่วยตรวจสอบให้ก็ได้ครับ

บทความเกี่ยวกับสิ่งพิมพ์อื่นๆ

  • ระบบสี CMYK กับ RGB ต่างกันอย่างไร
  • ส่ง FILES ARTWORK ให้โรงพิมพ์อย่างไร ให้ไม่มีปัญหา
  • นามสกุลไฟล์ ที่ใช้ในงานพิมพ์แต่ละชนิดแตกต่างกันยังไง
  • เทคนิคการทำไฟล์ Artwork ส่งโรงพิมพ์ Offset ให้ราคาถูกลง
  • ปัญหาที่พบบ่อยในงานสิ่งพิมพ์
  • คำแนะนำในการออกแบบ ฟอนต์ สี และการจัดรูปแบบของสื่อสิ่งพิมพ์
  • วิธีแก้ปัญหาสระเด้ง/ลอย ให้ถูกต้อง
  • ประโยชน์และวิธีการใช้ Pantone ที่ถูกต้อง
  • วิธีเตรียมไฟล์งานพิมพ์ให้สมบูรณ์แบบ
